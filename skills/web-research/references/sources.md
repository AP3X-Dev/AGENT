# Trusted Source Guide

## High-Trust Domains

### Official Documentation
- `.gov` - Government sources
- `.edu` - Educational institutions
- Company official docs (e.g., docs.python.org, developer.mozilla.org)

### Academic & Research
- arxiv.org - Preprints
- scholar.google.com - Academic papers
- IEEE, ACM - Peer-reviewed

### News & Tech
- reuters.com, apnews.com - Wire services
- arstechnica.com, theverge.com - Tech journalism
- hacker-news (comments often valuable for tech context)

## Medium-Trust Sources

- Wikipedia - Good for background, verify claims
- Stack Overflow - Technical solutions, check dates
- GitHub - Code examples, check stars/activity

## Citation Format Examples

### Inline Citation
```markdown
According to [Mozilla Developer Network](https://developer.mozilla.org),
the fetch API returns a Promise that resolves to the Response.
```

### Footnote Style
```markdown
The async/await syntax was introduced in ES2017[^1].

[^1]: MDN Web Docs - https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Statements/async_function
```

### Source List
```markdown
## Sources
1. [MDN: Fetch API](https://developer.mozilla.org/en-US/docs/Web/API/Fetch_API) - Official documentation
2. [Stack Overflow: Async patterns](https://stackoverflow.com/q/12345) - Community solution
```
